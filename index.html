Projects

Procedurally Generated World

In my first semester of year 3, I developed a procedurally generated world as a coursework project. It features terrain created with gradient noise and compute shaders, many different biomes that are laid out using cellular automata rules. It provides a huge landscape to explore that is dynamically created as you fly around.

This project was created using C++ and Direct3D 11. The source code for this project will be made public once grades have been released.

Crystal Caverns / SFMLEngine

Crystal Caverns is a game I developed as a coursework project for my first year of university. The main feature of this project is that it is created using my own advanced framework built on top of the existing SFML library.

SFMLEngine has a variety of advanced systems, including resource management, entity component systems, a sprite rendering system that supports custom shaders, physics and collision systems and more. Check out the README for more information.
